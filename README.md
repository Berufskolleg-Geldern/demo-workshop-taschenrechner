
# Taschenrechner App – Demo Projekt

Willkommen zum **Taschenrechner App** Projekt! Dieses Repository wurde als **Demo-Projekt** für einen **App-Entwicklungs-Workshop** entwickelt, der an unserer Schule stattfindet. In diesem Workshop lernen Schüler die Grundlagen der **App-Entwicklung** mit **React Native** und **Expo**.

## 📝 Projektbeschreibung

Das Ziel dieses Projekts ist es, eine einfache Taschenrechner-App zu erstellen, die grundlegende mathematische Operationen wie Addition, Subtraktion, Multiplikation und Division unterstützt. Die App dient als Lernplattform, um den Teilnehmern die praktischen Grundlagen der App-Entwicklung zu vermitteln.

## 📦 Anforderungen

Um dieses Projekt lokal auszuführen, benötigst du folgende Tools:
- **Node.js** (Version 18.x oder höher)
- **npm** (Paketmanager)
- Ein Mobilgerät oder Emulator (iOS oder Android)

## 🚀 Installation

Folge diesen Schritten, um das Projekt lokal auszuführen:

1. **Repository klonen:**
   ```bash
   git clone https://github.com/Berufskolleg-Geldern/demo-workshop-taschenrechner.git
   cd demo-workshop-taschenrechner
   ```

2. **Abhängigkeiten installieren:**
   ```bash
   npm install
   ```

3. **Entwicklungsserver starten:**
   ```bash
   npx expo start
   ```

4. **App auf dem Gerät ausführen:**
   - Scanne den QR-Code mit der **Expo Go App** (verfügbar für Android und iOS) oder
   - Starte den Emulator deiner Wahl (Android oder iOS).

## 📱 Features der App

- **Addition** zweier Zahlen
- **Subtraktion** zweier Zahlen
- **Multiplikation** zweier Zahlen
- **Division** zweier Zahlen
- Einfaches, benutzerfreundliches **User Interface** mit Buttons zur Eingabe der Zahlen und Operationen.
